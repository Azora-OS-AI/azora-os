## 🚀 GETTING STARTED

### Prerequisites

```bash
# Required
- Docker & Docker Compose
- Node.js 18+ LTS
- Git

# Recommended
- MetaMask wallet
- 8GB+ RAM
- 50GB+ disk space
```

### Quick Start (30 seconds)

```bash
# 1. Clone repository
git clone https://github.com/azora-os/azora-os.git
cd azora-os

# 2. Deploy everything
./DEPLOY.sh

# 3. Access platform
open http://localhost:5173
```

### Manual Setup

```bash
# Install dependencies
npm install

# Start infrastructure
docker-compose up -d

# Deploy smart contracts
cd azora-coin
npx hardhat compile
npx hardhat run scripts/deploy.js --network localhost

# Start services
npm run dev
```
