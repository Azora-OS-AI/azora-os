# Azora Nexus NFT Service

Non-Fungible Token (NFT) management service.

## Overview

Handles NFT creation, trading, and management operations.

## Features

- NFT minting
- NFT trading
- Collection management
- Metadata management
- Ownership tracking

## Installation

```bash
npm install
```

## Usage

```bash
npm start
```

## API Endpoints

### `GET /health`
Health check endpoint.

### `POST /api/nft/mint`
Mint new NFT.

### `GET /api/nft/:tokenId`
Get NFT information.
